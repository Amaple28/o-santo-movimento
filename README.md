# O Santo Movimento - Treinamento Funcional com Propósito

## 📋 Sobre o Projeto

**O Santo Movimento** é um site moderno e responsivo para um estúdio de treinamento funcional com propósito cristão. O site apresenta uma estética minimalista inspirada no Vidya Studio, com foco em acessibilidade, usabilidade e conversão.

### Características Principais

- ✅ Design minimalista e sofisticado
- ✅ Totalmente responsivo (mobile, tablet, desktop)
- ✅ Animações suaves e interativas
- ✅ Otimizado para SEO
- ✅ Paleta de cores limpa (Branco, Cinza, Laranja #FF6B35)
- ✅ Tipografia elegante (Playfair Display + Lato)
- ✅ Velocidade de carregamento otimizada
- ✅ Acessibilidade WCAG 2.1

## 📁 Estrutura de Arquivos

```
o-santo-movimento/
├── index.html          # Arquivo HTML principal
├── styles.css          # Estilos CSS (minimalista)
├── script.js           # JavaScript para animações

```

## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Baixe ou extraia os arquivos do projeto
2. Abra o arquivo `index.html` em seu navegador
3. Pronto! O site funcionará completamente offline

### Opção 2: Fazer Deploy Online
1. Faça upload dos arquivos para seu servidor web
2. Certifique-se de que `index.html`, `styles.css` e `script.js` estão no mesmo diretório
3. Acesse o site através da URL do seu domínio

### Opção 3: Usar com um Servidor Local (Python)
```bash
# Python 3
python -m http.server 8000

# Depois acesse: http://localhost:8000
```

## 🎨 Personalização

### Alterar Cores
Edite as variáveis CSS no topo do arquivo `styles.css`:

```css
:root {
    --primary-orange: #FF6B35;        /* Cor principal */
    --primary-orange-dark: #E55A24;   /* Cor hover */
    --white: #FFFFFF;                 /* Branco */
    --light-gray: #F5F5F5;            /* Cinza claro */
    --text-dark: #1A1A1A;             /* Texto escuro */
    --text-light: #666666;            /* Texto claro */
}
```

### Alterar Fontes
As fontes estão importadas do Google Fonts no `index.html`:
- **Títulos**: Playfair Display (serif)
- **Corpo**: Lato (sans-serif)

Para mudar, edite a seção de Google Fonts no `<head>` do HTML.
